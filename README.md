# 1- Axes d’analyse et tables de dimensions

## Axes d’analyse pertinents :

- Étudiant → analyser selon le sexe, la région, le statut (actif, diplômé…).

- Filière → analyser selon le nom de la filière, le niveau (Licence, Master, Doctorat).

Temps → analyser par année, semestre, ou mois d’inscription.

Inscriptions (fait) → mesurer le nombre d’inscriptions et le montant payé.

Tables du modèle étoile :

Fait_inscription : contient les mesures (montant payé, nombre d’inscriptions).

Dim_etudiant : informations sur les étudiants.

Dim_filiere : informations sur les filières.

Dim_temps : informations sur les dates (jour, mois, année).
